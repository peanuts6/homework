# Java位运算&基础类型





##
### 基本数据类型
 * char     - 2byte
 * boolean  - 2byte
 * byte     - 1byte
 * short    - 2byte
 * int      - 4byte
 * long     - 8byte
 * float    - 4byte
 * double   - 8byte

### 位运算
 * << 左移，右边补`0`
 * >> 右移，左边补`0`或`1`
 * >>> 无符号右移，左边补`0`
 * & 位与
 * | 位或
 * ~ 取反
 * ^ 异或 -- 不相同为1
